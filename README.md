## duck-street-runner
A javascript 3d duck runner using 3js. 
First, download it, **but do not download from the issues page**. Instead go to the code page and click on each files and click download raw file. 
Than, you can go to your file explorer and you create a folder for those downloaded files, but make sure the are in the same one. 
Next, open the html file in your browser, but if it doesn't load the css and javascript file, check the console and fix it. Normally because the it did not import those files, which you can try putting the css and js in to a subfolder.
Lastly, play the game and enjoy!
